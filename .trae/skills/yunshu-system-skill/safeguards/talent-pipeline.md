> 加载方式：01-init专项评估门禁触发

## 初级开发者保护规则

```
任务分配时：
  ↓
1. 评估任务难度（简单/中等/复杂）
  ↓
2. 简单任务 → 优先分配给初级开发者
  ↓
3. 中等任务 → 初级开发者主导，高级开发者审查
  ↓
4. 复杂任务 → 高级开发者主导，初级开发者参与学习
  ↓
5. 每个任务完成后：
   ├─ 初级开发者复述理解（理解验证）
   ├─ 高级开发者补充知识点（知识传承）
   └─ 记录到知识库（持久化）
```

## 反模式

- 🚫 所有任务都让AI做不给初级开发者机会（P-075入门路径）
- 🚫 高级开发者用AI编码不传授知识（P-077知识断裂）

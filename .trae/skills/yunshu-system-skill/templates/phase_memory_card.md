## 7) 上下文换挡
当你发现开始遗忘历史决策/反复问相同问题/需要引用大量历史时，执行：
1. 先运行 `python scripts/yunshu.py context record ...`，记录本阶段已读来源、关键发现、动作、缺口和下一轮补读
2. 再运行 `python scripts/yunshu.py checkpoint create ...`，写出本卡片（本文件）+ `.yunshu/checkpoints/<checkpoint_id>/phase_memory.json` + `.yunshu/checkpoints/<checkpoint_id>/checkpoint.json`
3. 在 Trae 新开对话（或清空当前对话）
4. 在新对话里用 `#File` 引用上述 2–3 个文件
5. 对话开头一句话恢复：**"使用云舒系统，从 checkpoint_id=<...> 恢复继续执行；以 phase_memory 为唯一历史事实来源，并先检查 context ledger 新鲜度。"**

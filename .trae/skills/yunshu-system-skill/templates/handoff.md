## 恢复指令

> 新会话中执行以下步骤即可恢复：

1. 读取本文件和 `phase_memory_card.md`
2. 确认 git 状态：`git status` + `git log --oneline -5`
3. 从未完成的第一个任务继续执行
4. 恢复后在本文件 frontmatter 标记 `restored: true`

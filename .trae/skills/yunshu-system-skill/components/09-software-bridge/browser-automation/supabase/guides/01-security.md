# 安全指南

---

## 密码存储规则

1. **唯一来源**：密码仅通过环境变量注入，禁止写入任何代码文件、配置文件、日志文件
2. **禁止传输**：禁止在聊天输出、截图标注、调试信息中展示明文密码
3. **生命周期**：密码仅在内存中驻留，进程结束后立即释放

---

## 环境变量使用规范

| 规范 | 说明 |
|------|------|
| 命名 | 统一使用 `SUPABASE_` 前缀，避免与其他服务冲突 |
| 加载时机 | 在技能激活时立即检查，缺失则阻断后续流程 |
| 文件位置 | `.env` 文件必须位于技能根目录或项目根目录，且已加入 `.gitignore` |
| 示例文件 | `.env.example` 仅包含占位值，禁止填入真实凭证 |

---

## 敏感信息保护

- **日志脱敏**：所有日志中的密码字段替换为 `***`
- **截图脱敏**：若截图包含密码输入框，确保密码已用圆点/星号掩码
- **2FA 隔离**：验证码由用户手动输入，禁止缓存或自动填充
- **会话安全**：浏览器会话 token 不持久化，关闭即失效

---

## 违规处理

检测到以下行为时立即终止执行并告警：
- 代码中出现硬编码密码字符串
- 日志输出包含明文密码
- 尝试将凭证写入磁盘文件

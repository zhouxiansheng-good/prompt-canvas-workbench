# 复古未来风格模板（Retro-futuristic）

> 美学方向：80年代科幻感，霓虹、网格
> 适用场景：科技类、游戏类、音乐、创意工具
> 参考：Synthwave、Cyberpunk 2077、Blade Runner

---

## 设计特点

- **霓虹色彩** — 粉、蓝、紫、绿
- **网格背景** — 透视网格、扫描线
- **发光效果** — 文字发光、边框发光
- **科技字体** — 等宽字体、未来感字体
- **深色主题** — 黑色/深蓝背景

## 排版

- **展示字体**：Orbitron / Rajdhani / 等宽字体
- **正文字体**：Rajdhani / Exo 2
- **发光文字** — text-shadow 发光效果
- **扫描线效果** — 文字上的扫描线动画

## 色彩

- **深色背景** — 深蓝 #0A0E27 / 黑色 #000000
- **霓虹粉** — #FF00FF
- **霓虹蓝** — #00FFFF
- **霓虹紫** — #9900FF
- **霓虹绿** — #00FF00

```css
--color-background: #0A0E27;
--color-surface: #1A1F3A;
--color-neon-pink: #FF00FF;
--color-neon-blue: #00FFFF;
--color-neon-purple: #9900FF;
--color-neon-green: #00FF00;
--color-text: #FFFFFF;
--color-text-dim: #8B92B9;
```

## 布局

- **网格背景** — 透视网格线
- **不对称** — 打破对称
- **HUD 风格** — 类似游戏界面
- **分屏** — 左右分屏对比

## 动效

- **扫描线** — 屏幕扫描线效果
- **闪烁** — 霓虹灯闪烁
- **故障效果** — Glitch 效果
- **音频频谱** — 音频可视化

## 背景

- **星空/网格** — 透视网格或星空
- **扫描线叠加** — 屏幕扫描线
- **噪点纹理** — 复古噪点

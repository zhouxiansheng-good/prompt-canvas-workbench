# 极简风格模板（Minimal）

> 美学方向：极度简洁，留白为主，克制
> 适用场景：工具类、专业应用、SaaS
> 参考：Apple、Notion、Linear

---

## 设计特点

- **大量留白** — 元素之间呼吸感强
- **精简色彩** — 1-2 种主色，无渐变
- **清晰层次** — 通过字号和间距建立层次
- **功能优先** — 装饰元素极少
- **精确对齐** — 严格的网格和对齐

## 排版

- **展示字体**：SF Pro Display / Helvetica Neue（系统级）
- **正文字体**：SF Pro Text / Inter（极简例外：允许使用 Inter）
- **字号体系**：
  - 大标题：48-64px，字重 700
  - 小标题：24-32px，字重 600
  - 正文：16px，字重 400
  - 辅助：14px，字重 400

## 色彩

- **背景**：纯白 #FFFFFF 或 极浅灰 #F5F5F5
- **主色**：黑色 #000000 或 深灰 #1A1A1A
- **强调色**：单一强调色（蓝/绿/橙），极少使用
- **文字**：深灰 #1A1A1A / 中灰 #666666 / 浅灰 #999999

```css
--color-background: #FFFFFF;
--color-surface: #F5F5F5;
--color-primary: #000000;
--color-accent: #0066CC; /* 可替换 */
--color-text: #1A1A1A;
--color-text-secondary: #666666;
--color-border: #E5E5E5;
```

## 布局

- **对称网格** — 极简风格允许对称
- **大量留白** — 模块间距 64-128px
- **单列为主** — 内容区宽度 640-800px
- **固定导航** — 简洁的顶部导航

## 动效

- **极简动效** — 淡入淡出（极简例外：允许简单淡入）
- **微交互** — 悬停状态微妙变化
- **页面过渡** — 快速、不显眼

## 背景

- **纯色为主** — 白色或极浅灰
- **无纹理** — 保持干净
- **可选细线分隔** — 1px #E5E5E5

## 差异化建议

- 独特的留白比例
- 精致的微交互
- 非标准的网格比例（如黄金分割）

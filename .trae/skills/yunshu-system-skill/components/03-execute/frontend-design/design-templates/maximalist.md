# 极繁风格模板（Maximalist）

> 美学方向：丰富、混乱、充满活力，大胆
> 适用场景：创意类、展示类、艺术、时尚
> 参考：Gucci、Museum websites、Dazed

---

## 设计特点

- **元素丰富** — 大量视觉元素
- **大胆色彩** — 多种鲜艳色彩碰撞
- **复杂排版** — 多种字体混排
- **重叠元素** — 打破常规层次
- **强烈对比** — 大小、色彩、粗细对比

## 排版

- **展示字体**：Clash Display / Syne / 装饰性字体
- **正文字体**：Space Grotesk / Manrope
- **多种字体混排** — 3-4 种字体同时使用
- **字号对比强烈** — 超大标题（120px+）+ 小正文

## 色彩

- **多色碰撞** — 3-5 种鲜艳色彩
- **高饱和度** — 不畏惧鲜艳色彩
- **渐变叠加** — 多层渐变叠加
- **霓虹色** — 粉、绿、蓝、橙

```css
--color-background: #FF0066;
--color-surface: #00FF88;
--color-primary: #000000;
--color-accent-1: #FF0066;
--color-accent-2: #00FF88;
--color-accent-3: #6600FF;
--color-text: #FFFFFF;
--color-text-inverse: #000000;
```

## 布局

- **打破网格** — 元素随意放置
- **重叠** — 图片重叠文字，文字重叠图片
- **不对称** — 左右不均等
- **全出血** — 图片延伸到边缘
- **多栏混排** — 不同宽度的栏目

## 动效

- **复杂动画** — 多个元素同时动画
- **滚动触发** — 滚动时元素飞入
- **悬停放大** — 图片悬停放大
- **视差效果** — 多层滚动速度不同

## 背景

- **鲜艳纯色** — 大胆的纯色背景
- **渐变叠加** — 多色渐变
- **图案纹理** — 几何图案、噪点

## 差异化建议

- 独特的色彩组合
- 大胆的排版实验
- 创新的交互方式

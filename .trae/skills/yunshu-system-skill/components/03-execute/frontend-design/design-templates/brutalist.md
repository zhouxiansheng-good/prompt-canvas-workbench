# 野兽派风格模板（Brutalist）

> 美学方向：原始、粗野、不加修饰、raw
> 适用场景：艺术类、独立品牌、实验项目、个人网站
> 参考：Brutalist Websites、Dezeen、Rick Owens

---

## 设计特点

- **原始材质** — 混凝土、金属、纸张纹理
- **粗野排版** — 超大字体、重叠、裁剪
- **高对比** — 黑白对比、强烈视觉冲击
- **无装饰** — 无圆角、无阴影、无渐变
- **系统默认** — 使用系统字体（野兽派例外）

## 排版

- **展示字体**：系统默认字体（Arial/Helvetica）
- **正文字体**：系统默认字体
- **超大字号** — 标题 100px+
- **文字重叠** — 文字重叠图片
- **文字裁剪** — 文字被容器裁剪

## 色彩

- **黑白为主** — 黑色 #000000 / 白色 #FFFFFF
- **单一强调色** — 红/黄/蓝（极少使用）
- **无渐变** — 纯色块
- **高对比** — 黑白对比

```css
--color-background: #FFFFFF;
--color-surface: #000000;
--color-primary: #000000;
--color-accent: #FF0000; /* 极少使用 */
--color-text: #000000;
--color-text-inverse: #FFFFFF;
--color-border: #000000;
```

## 布局

- **无网格** — 随意放置
- **重叠** — 元素重叠
- **全出血** — 图片延伸到边缘
- **不规则** — 不规则的形状和排列
- **大量留白** — 或极度拥挤

## 动效

- **无动效** — 或极简动效
- **直接切换** — 无过渡
- **滚动触发** — 简单的显示/隐藏

## 背景

- **纯色** — 白色或黑色
- **纹理** — 混凝土、纸张、噪点
- **无装饰** — 无渐变、无图案

## 差异化建议

- 独特的排版实验
- 原始材质纹理
- 大胆的留白或拥挤

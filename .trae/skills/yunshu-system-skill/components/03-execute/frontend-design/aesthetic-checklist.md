# 美学合规检查清单（Aesthetic Checklist）

> 解决P-015业务理解缺失：前端任务的设计质量维度
> 触发条件：03-execute Step 2.4 代码自检，前端任务时自动执行

---

## 检查流程

```
代码自检时
    ↓
判断是否为前端任务
    ├── 是 → 执行美学合规检查
    │         ↓
    │       逐项检查（7项）
    │         ↓
    │       全部通过？
    │         ├── 是 → 标记美学合规 ✅
    │         └── 否 → 记录问题 → 修复 → 重新检查
    └── 否 → 跳过美学检查
```

## 检查项

### 1. 字体合规（Typography Compliance）

**检查内容**：
- [ ] 未使用禁止字体（Inter / Roboto / Arial / 系统默认字体）
- [ ] 使用了设计决策中指定的字体
- [ ] 展示字体和正文字体有明确区分
- [ ] 字号体系一致（标题/正文/辅助）

**通过标准**：
- 无禁止字体
- 字体选择符合设计决策
- 至少使用了 2 种字体（展示 + 正文）

**失败处理**：
- 替换禁止字体为设计决策指定字体
- 若未指定，从 Google Fonts 选择特色字体

---

### 2. 色彩合规（Color Compliance）

**检查内容**：
- [ ] 未使用禁止色彩方案（紫色渐变 + 白色背景）
- [ ] 使用了设计决策中指定的色彩方案
- [ ] 使用了 CSS 变量管理色彩
- [ ] 色彩对比度符合 WCAG 要求

**通过标准**：
- 无紫色渐变 + 白背景组合
- 主色调和强调色明确
- 所有颜色通过 CSS 变量定义

**失败处理**：
- 替换禁止色彩为设计决策指定色彩
- 添加 CSS 变量统一管理

---

### 3. 布局合规（Layout Compliance）

**检查内容**：
- [ ] 布局符合设计决策（不对称/重叠/对角线等）
- [ ] 未使用标准 12 列网格（如选择了打破网格）
- [ ] 空间构图有特点（负空间/重叠/打破网格）
- [ ] 响应式适配完整

**通过标准**：
- 布局特点符合设计方向
- 非可预测的对称布局（除非极简风格）
- 移动端适配正常

**失败处理**：
- 调整布局实现设计决策中的特点
- 添加响应式断点

---

### 4. 动效合规（Motion Compliance）

**检查内容**：
- [ ] 实现了设计决策中的核心动效
- [ ] 动效性能符合要求（> 60 FPS）
- [ ] 动效风格一致（不混用不同动画库）
- [ ] 有页面加载动画或交互反馈

**通过标准**：
- 至少实现 1 个核心动效
- 动画流畅无卡顿
- 动效风格统一

**失败处理**：
- 补充核心动效
- 优化动画性能（使用 transform/opacity）

---

### 5. 背景合规（Background Compliance）

**检查内容**：
- [ ] 背景创造了氛围和深度
- [ ] 非纯色背景（除非极简风格）
- [ ] 有纹理/渐变/图案等效果
- [ ] 背景与整体风格一致

**通过标准**：
- 背景有层次感
- 符合设计方向（复古未来 → 网格/星空；有机自然 → 渐变/纹理）

**失败处理**：
- 添加背景效果（渐变/纹理/图案）
- 确保背景与风格一致

---

### 6. 差异化合规（Differentiation Compliance）

**检查内容**：
- [ ] 有独特记忆点（令人难忘的元素）
- [ ] 与常见设计有明显区别
- [ ] 有创新的交互或视觉元素
- [ ] 不是"又一个普通界面"

**通过标准**：
- 至少 1 个令人难忘的元素
- 与"AI风格"（Inter + 紫渐变）明显不同

**失败处理**：
- 增加独特设计元素
- 重新思考差异化策略

---

### 7. 一致性合规（Consistency Compliance）

**检查内容**：
- [ ] 整体风格统一（所有元素符合选定美学方向）
- [ ] 色彩体系一致
- [ ] 字体使用一致
- [ ] 间距体系一致
- [ ] 动效风格一致

**通过标准**：
- 无明显风格冲突
- 所有元素和谐统一
- 视觉层次清晰

**失败处理**：
- 统一不一致的元素
- 调整冲突的设计风格

## 常见失败模式

| 失败模式 | 原因 | 修复建议 |
|---------|------|---------|
| 使用了 Inter 字体 | 默认习惯 | 替换为设计决策指定字体 |
| 紫色渐变背景 | AI 默认倾向 | 替换为设计决策指定色彩 |
| 标准 12 列网格 | 习惯性布局 | 尝试不对称/重叠布局 |
| 缺少动效 | 忽略动画 | 添加核心动效（加载/悬停） |
| 纯色背景 | 图省事 | 添加渐变/纹理/图案 |
| 无独特元素 | 缺乏差异化思考 | 增加记忆点设计 |
| 风格不统一 | 混合多种风格 | 统一为选定美学方向 |

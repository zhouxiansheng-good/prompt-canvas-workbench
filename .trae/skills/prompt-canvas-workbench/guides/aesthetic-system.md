# 美术设计系统指南

## 设计哲学

复古拟物化工作台美学。参考色彩心理学，四套主题均经过明度和对比度调校，护眼舒适，适合长时间浏览。

## 四色主题

### 1. 经典复古 (paper)
- **心理暗示**: 安全、沉浸、天然
- **背景**: 牛皮纸色 `#eae6df`
- **卡片**: 纯白 `#ffffff`
- **文字**: 石墨黑 `#1c1c1c`
- **边框**: 深灰 `#2c2c2c`

### 2. 包豪斯墨绿 (blackboard)
- **心理暗示**: 缓解视觉疲劳、高度理性
- **背景**: 黑板绿 `#12241c`
- **卡片**: 深绿 `#1a3328`
- **文字**: 粉蜡乳白 `#f0fdf4`
- **边框**: 浅绿 `#e8f5e9`

### 3. 北欧极夜 (nordic)
- **心理暗示**: 克制、冰冷、未来感
- **背景**: 极夜深蓝灰 `#0b0f19`
- **卡片**: 深蓝 `#111827`
- **文字**: 冷白 `#f9fafb`
- **边框**: 雾霾蓝 `#38bdf8`

### 4. 多巴胺柔和派 (dopamine)
- **心理暗示**: 愉悦、轻松、打破思维框架
- **背景**: 马卡龙粉蓝 `#e0f2fe`
- **卡片**: 纯白 `#ffffff`
- **文字**: 深紫 `#4c1d95`
- **边框**: 紫色 `#4c1d95`

## 阴影系统

| 名称 | 值 | 用途 |
|------|-----|------|
| retro-sm | 2px 2px 0px 0px | 小按钮、标签 |
| retro-md | 5px 5px 0px 0px | 卡片默认 |
| retro-lg | 10px 10px 0px 0px | 卡片hover、弹窗 |

## 字体系统

- **衬线体**: Noto Serif SC, Songti SC（标题、装饰文字）
- **无衬线体**: Inter, PingFang SC（正文）
- **等宽体**: Fira Code（代码、数据）

## 网格背景

24px 网格线，使用 CSS `linear-gradient` 实现，透明度极低（0.04-0.06），不干扰内容阅读。

## 响应式策略

- **桌面端 (>=1024px)**: 固定320px侧边栏，卡片330px宽，overflow:hidden
- **移动端 (<1024px)**: 抽屉式侧边栏，卡片自适应宽度，overflow-x:hidden
